# NetHack Babel — 玩家指南

[English Guidebook](GUIDEBOOK.md)

## 1. 简介

NetHack Babel 是 NetHack 3.7 的忠实 Rust 重实现，覆盖原版 99.8% 的游戏系统，4,217 个测试验证。如果你玩过 NetHack，一切机制都与你熟悉的相同——相同的公式、相同的技巧、相同的死法。如果你是新手，欢迎来到有史以来最深邃的 Roguelike 游戏之一。

你的目标：深入厄运地牢（Dungeons of Doom），从最底层取回延多的护身符（Amulet of Yendor），穿越元素位面上升，在星界的正确祭坛上献祭护身符。你会死很多次才能成功。这是正常的。

## 2. 开始游戏

### 启动

```sh
cargo run -- --data-dir data --language zh_CN      # 简体中文
cargo run -- --data-dir data --language zh_TW      # 繁体中文
cargo run -- --data-dir data                       # 英文
cargo run -- --replay ./session.cast               # 回放录制会话
cargo run -- --server 127.0.0.1:2323               # 启动 TCP 服务器模式（MVP）
```

游戏内按 `O` 可随时切换语言，无需重启。

### 屏幕布局

```
  消息显示在这里（顶部行）
  ┌──────────────────────────────────────┐
  │  #####                               │
  │  #...#         地图（80×21）          │
  │  #.@.#         @ = 你                │
  │  #...#         . = 地板              │
  │  ###+####      # = 墙壁 / 走廊       │
  │  #......#      + = 门                │
  │  ########      < > = 楼梯            │
  └──────────────────────────────────────┘
  状态栏 1：名字  力:18 敏:14 体:16 智:10 感:8  魅:12
  状态栏 2：层:1 $:0 HP:16(16) Pw:4(4) AC:10 经验:1 T:1
```

### 你的第一局

1. 用 `hjkl`（vi 键）或方向键移动。对角移动：`yubn`。
2. 走向怪物即可攻击。
3. 按 `,` 拾取物品。按 `i` 查看背包。
4. 饿了按 `e` 吃东西。按 `q` 饮用药水。
5. 在 `>` 格子上按 `>` 下楼。
6. 按 `S` 保存游戏。

## 3. 移动与动作

### 基础移动

| 按键 | 方向 | 按键 | 方向 |
|------|------|------|------|
| `h` | 西 | `l` | 东 |
| `j` | 南 | `k` | 北 |
| `y` | 西北 | `u` | 东北 |
| `b` | 西南 | `n` | 东南 |
| `.` | 等待（休息） | `s` | 搜索（寻找隐藏门/陷阱） |

**Shift + 方向** = 持续移动直到被打断（奔跑）。穿越长走廊时很有用。

**计数前缀**：在命令前输入数字以重复执行。`20s` = 搜索 20 次。

### 楼梯

- 在 `<` 格子上按 `<`：上楼
- 在 `>` 格子上按 `>`：下楼

地牢会记住你访问过的每一层。你留下的怪物在你返回时仍在原处。

## 4. 战斗

### 近战

走向怪物即可攻击。命中概率取决于：

- 你的经验等级和属性（力量、敏捷）
- 武器的附魔值和技能等级
- 目标的护甲等级（AC 越低越难命中）
- 你的运气值（范围 -13 到 +13）

### 武器技能

随着战斗你会提升熟练度：受限 → 基础 → 精通 → 专家。更高技能意味着更好的命中和伤害加成。按 `#enhance` 使用技能点。

### 重要战斗机制

- **负 AC**：当怪物 AC 低于 0 时，需要通过额外的判定才能命中
- **银制武器**：对惧银怪物（恶魔、亡灵）额外造成 d20 伤害
- **祝福武器**：对亡灵和恶魔额外造成 d4 伤害
- **背刺**：盗贼在隐身状态从背后攻击时获得额外伤害
- **武僧**：空手战斗最强——穿甲会施加严厉的 -20 命中惩罚

## 5. 物品

### 物品类型

| 符号 | 类型 | 使用键 |
|------|------|--------|
| `)` | 武器 | `w` 装备 |
| `[` | 护甲 | `W` 穿戴，`T` 脱下 |
| `!` | 药水 | `q` 饮用 |
| `?` | 卷轴 | `r` 阅读 |
| `/` | 魔杖 | `z` 使用 |
| `=` | 戒指 | `P` 佩戴，`R` 摘下 |
| `"` | 护身符 | `P` 佩戴，`R` 摘下 |
| `(` | 工具 | `a` 使用 |
| `%` | 食物 | `e` 吃 |
| `+` | 魔法书 | `r` 阅读（学习） |
| `*` | 宝石 | `,` 拾取 |
| `$` | 金币 | 自动拾取 |

### BUC 状态（祝福 / 未诅咒 / 被诅咒）

每件物品都有 BUC 状态，影响其效果：

- **祝福**（B）：增强效果。祝福的治疗药水恢复更多 HP，祝福的鉴定卷轴鉴定全背包。
- **未诅咒**（U）：正常效果。
- **被诅咒**（C）：削弱或反转效果。被诅咒的升级药水会*失去*一级。被诅咒的装备无法脱下。

将物品浸入圣水（祝福的清水）可以祝福它们。祭坛可以通过放下物品来测试 BUC 状态。

### 鉴定

物品初始未鉴定。鉴定方法：
- **直接使用** — 喝下那瓶未知药水（有风险！）
- **鉴定卷轴** — 安全但数量有限
- **价格鉴定** — 商店价格可以推断物品身份（经典策略）
- **祭坛测试** — 在祭坛上放下物品可以揭示 BUC 状态

### 中文物品命名

使用中文游玩时，物品显示符合中文语法：

| 英文 | 简体中文 |
|------|---------|
| a long sword | 长剑 |
| a blessed +2 long sword | 祝福的+2长剑 |
| 3 daggers | 3把匕首 |
| a potion of healing | 治疗药水 |

量词自动应用：把（武器）、瓶（药水）、张（卷轴）等。

## 6. 怪物

### 怪物速度

怪物和玩家基于移动点数行动。速度更快的怪物每回合获得更多行动次数。同速怪物按创建顺序行动（先到先得）。

### 怪物 AI

怪物并不无脑：
- **智能怪物** 会开门、使用魔杖、穿戴装备、饮用药水
- **受伤怪物**（HP < 25%）可能逃跑
- **觊觎怪物**（巫师）会传送过来偷走关键物品
- **飞行怪物** 可以跨越水面和熔岩
- **穿墙怪物** 可以穿过墙壁

### 宠物

你的初始宠物（猫或狗）会跟随你、为你战斗、拾取物品。保持喂食以维持忠诚度——挨饿的宠物会变成野生。宠物只有在你上下楼时与你相邻才能跟随。

**经典策略**：宠物可以站在商店物品上然后被你挤出商店，从而"偷走"物品。

## 7. 地牢

### 地牢分支

| 分支 | 深度 | 特点 |
|------|------|------|
| 主地牢 | 1-29 | 向下的主要路径 |
| 侏儒矿山 | 入口 3-5 层 | 矿镇、商店、宝石 |
| 推箱子（Sokoban） | 从矿山进入 | 推箱子谜题，保证有好装备 |
| 任务 | 游戏中期 | 职业专属，神器奖励 |
| 卢迪奥斯堡垒 | 随机 | 充满金币的金库 |
| 冥界（Gehennom） | 城堡以下 | 恶魔、迷宫、无法祈祷 |
| 弗拉德之塔 | 冥界内 | 召唤烛台 |
| 终局 | 上升后 | 元素位面 + 星界 |

### 特殊房间

房间可能是：商店、神殿、王座室、动物园、军营、蜂巢、太平间、沼泽、小妖精大厅、蛇发女妖巢穴、蚁穴。

### 陷阱

25 种陷阱潜伏在地牢中。按 `s` 搜索隐藏陷阱。一些重要陷阱：
- **坑 / 尖刺坑** — 坠落伤害，可能中毒
- **传送陷阱** — 将你传送到随机位置
- **变形陷阱** — 将你变成随机生物
- **反魔法场** — 消耗你的魔力
- **地雷** — 大量伤害和眩晕

## 8. 饥饿与进食

你每回合消耗 1 点营养，佩戴戒指、再生能力和延多护身符会额外消耗。饥饿等级：

| 等级 | 营养值 | 效果 |
|------|--------|------|
| 饱食 | > 1000 | 继续吃有窒息风险 |
| 不饿 | 150-1000 | 正常 |
| 饥饿 | 50-150 | 警告消息 |
| 虚弱 | 1-50 | 力量 -1，战斗受损 |
| 昏厥 | ≤ 0 | 随机昏厥，濒临死亡 |
| 饿死 | 深度负值 | 死亡 |

**尸体** 是主要食物来源，但有风险：变质的尸体会导致食物中毒，某些尸体赋予内在能力（漂浮之眼的心灵感应、杀人蜂的毒素抵抗）。

## 9. 宗教与祈祷

### 祈祷

在危急时刻按 `#pray`。你的神可能会：
1. 修复受损属性
2. 治愈疾病
3. 结束饥饿
4. 赐予有用物品

祈祷有冷却时间。成功与否取决于你的阵营记录、运气，以及是否在同阵营祭坛上。**在冥界（Gehennom）祈祷永远不会成功。**

### 献祭

杀死怪物后在祭坛上用 `#offer` 献祭尸体。好处：
- 提升阵营值和运气
- 获得神器赐予（第一件神器有保底）
- 转换祭坛阵营（有风险）

### 运气

运气范围 -13 到 +13。影响战斗、祈祷和许多随机事件。运气每 600 回合自然衰减向 0。携带幸运石（祝福或未诅咒）防止衰减。

## 10. 商店

店主会追踪一切。在商店里拿起物品就欠了钱。按 `p` 付款。

**定价** 取决于：基础价格 × BUC 修正 × 你的魅力 × 旅行者加价。

**价格鉴定**：精明的玩家可以通过店主的报价推断物品身份。这是 NetHack 的核心策略之一。

**盗窃**：不付款就离开商店，店主会攻击你。基石警察可能被召唤。

## 11. 状态效果

| 状态 | 持续时间 | 治愈方法 |
|------|---------|---------|
| 混乱 | 计时 | 等待、独角兽角 |
| 失明 | 计时 | 等待、胡萝卜、独角兽角 |
| 眩晕 | 计时 | 等待、独角兽角 |
| 幻觉 | 计时 | 等待、独角兽角 |
| 漂浮 | 计时 | 等待（无法走楼梯！） |
| 石化 | 5 回合 | 吃蜥蜴尸体或酸性食物 |
| 黏化 | 10 回合 | 用火 |
| 食物中毒 | 倒计时 | 祈祷、独角兽角 |

**独角兽角** 是最万能的治愈工具——一次使用可修复混乱、眩晕、幻觉和失明（祝福的必定治愈，未诅咒约 33% 概率，被诅咒永远不治愈）。

## 12. 经典策略

这些久经考验的策略在 NetHack Babel 中全部有效：

- **Elbereth**：刻写"Elbereth"吓退大部分怪物。用火焰魔杖烧刻可使其永久。
- **漂浮之眼 + 心灵感应**：吃漂浮之眼尸体获得永久心灵感应。隔墙探测怪物的必备能力。
- **Excalibur 浸泡**：守序阵营且至少 5 级时，将长剑浸入喷泉获得圣剑 Excalibur。
- **果冻养殖**：用负附魔武器攻击棕色/黑色果冻使其分裂。重复获得无限尸体和经验。
- **价格鉴定**：利用店主报价鉴定药水、卷轴和其他物品，无需冒险。
- **宠物偷窃**：让宠物站在商店物品上，然后将其挤出商店。宠物"偷走"物品给你。
- **独角兽角**：使用独角兽角治愈几乎所有负面状态效果。

## 13. 操守（Conduct）

NetHack 追踪称为"操守"的自愿挑战。在整局游戏中保持某项操守是实力的标志：

| 操守 | 规则 |
|------|------|
| 不进食 | 从不吃东西 |
| 纯素 | 从不吃动物制品 |
| 素食 | 从不吃肉 |
| 无神论 | 从不祈祷或献祭 |
| 不用武器 | 从不在战斗中装备武器 |
| 和平主义 | 从不直接杀死怪物 |
| 文盲 | 从不阅读卷轴或魔法书 |
| 不灭绝 | 从不使用灭绝 |
| 不变形物品 | 从不变形物品 |
| 不变形自身 | 从不变形自己 |
| 不许愿 | 从不许愿获取物品 |
| 不许愿神器 | 从不许愿获取神器 |
| 不刻 Elbereth | 从不写 Elbereth |

## 14. 巫师模式（调试）

使用 `-D` 参数启动游戏进入巫师模式：

```sh
cargo run -- --data-dir data -D
```

巫师模式命令：

| 按键 | 命令 | 效果 |
|------|------|------|
| Ctrl+W | 许愿 | 许愿获得任何物品（如 "blessed +3 silver dragon scale mail"） |
| Ctrl+F | 地图 | 揭示当前关卡全貌 |
| Ctrl+G | 创建 | 按名称创建怪物 |
| Ctrl+I | 鉴定 | 鉴定背包中所有物品 |
| Ctrl+E | 探测 | 揭示所有怪物、物品和陷阱 |
| `#levelchange` | 关卡传送 | 跳转到指定地牢深度 |
| `#where` | 位置 | 显示所有特殊关卡的位置 |

巫师模式对于测试游戏机制和探索内容极为有用，不受永久死亡的压力。

## 16. 新手建议

1. **别什么都吃** — 变质的尸体会致死。用 `;`（查看）检查时间。
2. **鉴定后再用** — 未知药水和卷轴可能致命。
3. **留一根独角兽角** — 游戏中最好的状态治愈工具。
4. **危急时祈祷** — 但不要太频繁，也永远不要在冥界祈祷。
5. **在商店做价格鉴定** — 学习价格表，安全鉴定物品。
6. **不要什么都打** — 善用 Elbereth、走廊和门进行战术。
7. **宠物很有价值** — 喂饱的宠物为你战斗，还能偷商店东西。
8. **省着用鉴定卷轴** — 用在高价值的未鉴定物品上。
9. **随身带一具蜥蜴尸体** — 永不腐烂，治愈石化。
10. **开发团队想到了一切** — 如果你想"我能不能……"，答案很可能是"能"。

## 17. 原版 NetHack 速览

原版 NetHack 是由 NetHack DevTeam 长期维护的经典 Roguelike 分支，核心特征是程序化地牢、极强的系统交互深度和严格的永久死亡机制。它既是策略游戏，也是系统模拟游戏：背包管理、站位、运气值、时机选择都会影响整局成败。

NetHack Babel 保持了这条经典主线：

1. 深入厄运地牢及各分支。
2. 取回延多护身符。
3. 穿越元素位面向上攀升。
4. 在星界正确祭坛献祭护身符。

因此，原版 NetHack 的绝大多数经验和攻略在 Babel 中依然成立。
